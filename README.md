# Code Quality Dashboard

**Real-time LLM code quality analysis**
---
### What It Does
- **Simulates 100+ AI-generated Python functions** (like an LLM output)
- **Analyzes instantly**
- **Calculates lines of code per function**
- **Groups into quality tiers**:
  - **Low** (≤10 lines) – simple, maintainable
  - **Medium** (11–30 lines) – balanced
  - **High** (>30 lines) – complex, needs reviewing
- **Shows metrics + bar chart**
---
### How to Run
```bash
pip install streamlit pandas numpy
streamlit run main.py
```
